# Pipeline Dashboard Backend MVP

Utility #30 of RADAR_SERVICE.

## Purpose

Provide an artifact-first backend for loading, normalizing, aggregating,
querying, and serving runtime observability data produced by RADAR_SERVICE
utilities.

## Initial Input Sources

- Pipeline Metrics Report
- Pipeline Health Report
- Pipeline Health Trend Report
- Runtime Event Report

## MVP Responsibilities

- Load observability artifacts
- Normalize dashboard data
- Aggregate runtime state
- Provide query services
- Build dashboard snapshots
- Expose API and CLI outputs

## Non-Goals

- Frontend dashboard rendering
- Pipeline execution
- Source artifact mutation
- Retry or recovery orchestration
- Runtime intelligence decisions
